# Prasangi Dance Studio & Fitness Center Website

## Overview

This project is a modern and responsive website developed for **Prasangi Dance Studio & Fitness Center**, located in **Jammikunta**. The website provides detailed information about the dance studio, available dance styles, contact information, and an interactive gallery showcasing the studio’s activities and achievements.

The goal of this project was to create a visually engaging and user-friendly platform that helps students and parents easily explore dance classes and connect with the studio.

---

## Features

* **3D Animated Hero Section** with an interactive dancer model
* **Responsive Design** optimized for both desktop and mobile devices
* **About Section** introducing the dance studio and its mission
* **Dance Styles Section** showcasing different dance forms offered
* **Gallery Section** with image preview and lightbox effect
* **Contact Form Integration** using EmailJS to send enquiries directly to the studio email
* **Google Maps Integration** showing the studio location
* **Floating WhatsApp Contact Button** for quick communication
* **Smooth Scrolling Navigation**
* **SEO Meta Tags** for better search engine visibility
* **Deployed Live Website** using Vercel

---

## Technologies Used

* **React.js**
* **Vite**
* **CSS3**
* **Framer Motion** (for animations)
* **React Three Fiber / Three.js** (for 3D model rendering)
* **EmailJS** (for contact form email service)
* **Lucide React Icons**
* **Git & GitHub**
* **Vercel** (deployment)
